## ZMX Caretaker Bot v1 (Discord)

- `Уведомления Discord`: Автоматически отправляет уведомления о важных событиях, включая сборы в Telegram-чатах, прямо в каналы Discord.
- `Webhook API`: Поддерживает интеграцию через вебхуки для получения информации о событиях из внешних систем.
- `Форматированные сообщения`: Генерирует красиво оформленные сообщения с упоминанием пользователей или ролей, деталями события и временем объявления.
- `Надёжность подключения`: Реализован механизм автоматического переподключения к Discord при обрыве соединения.
- `Поддержка ролей и пользователей`: Гибкая работа с упоминаниями как отдельных пользователей, так и ролей через идентификаторы.
- `Лёгкая настройка`: Настройки, такие как токен бота и ID канала, задаются через файл .env.
- `Безопасность`: Обрабатывает входящие запросы безопасно, проверяя наличие необходимых параметров.
- `Масштабируемость`: Возможность добавления новых маршрутов вебхуков для обработки различных типов уведомлений.
